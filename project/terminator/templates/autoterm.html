{% blocktrans %}Autoterm is a feature of the free software localization tool Virtaal. It consists in automatically getting from the Internet a TBX file with terminology data for the language Virtaal currently is using for translating to, without no need for the user doing nothing in order to getting it. In order to do that Virtaal has a list of URLs from which it can retrieve a TBX file for a language.{% endblocktrans %}

{% blocktrans %}The Terminator Autoterm feature is aimed to provide this TBX files on the fly by visiting some specific URLs, with the latest changes and improvements done to the exported data. This way Terminator will allow to Virtaal only store a list of URLs where Terminator instances are running instead of listing a lot of URLs. By default it exports data from all the glossaries in the Terminator instance, and only for the chosen language and English. It also has other constraints, like exporting only finalized definitions, and only finalized and preferred translations.{% endblocktrans %}
